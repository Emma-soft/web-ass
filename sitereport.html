How It All Began...

September 2020.

---Newbie to computer science with little knowledge on how programming works, I had to make some extra moves in order to be in the same level as my mates.

As difficult as it seamed, I had to make it work and being the kind of person that I am, there was not going to be any lose ends. I had to watch the class modules at least twice for every activity and also try out same code as often as possible just to be able to understand the concepts of coding and its principles

By December, I was already feeling good about myself, I can now do a few things with my comouter, and had also been able to test a few things without going back to the class lecture. Or So I Thought.
